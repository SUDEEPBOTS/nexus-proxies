# Contributing to Nexus Proxies

First off, thank you for considering contributing to Nexus Proxies! 

## How to Contribute

### 1. Report Bugs
If you find a bug, please create an issue with:
* A clear and descriptive title
* Steps to reproduce the bug
* Expected and actual behavior
* Your browser and OS

### 2. Suggest Enhancements
We welcome ideas for new features! Please open an issue to discuss it before you start coding to ensure it aligns with the project's goals.

### 3. Pull Requests
* **Fork** the repository and create a new branch (`git checkout -b feature/awesome-feature`).
* **Develop** your feature or fix.
* **Test** your changes locally.
* **Commit** your changes with a clear commit message (`git commit -m 'Add awesome feature'`).
* **Push** to the branch (`git push origin feature/awesome-feature`).
* **Open a Pull Request** describing your changes.

## Development Setup
```bash
npm install
npm run dev
```

Thank you for making Nexus Proxies better!
